# Tutorial GitHub y Docker

## Descripción
Este repositorio fue creado con el objetivo de aprender a crear y configurar una cuenta en GitHub.

## ¿Qué es GitHub?
GitHub es una plataforma que permite almacenar, administrar y compartir proyectos de software. Facilita el trabajo colaborativo.

## ¿Qué es Docker?
Docker es una herramienta que permite crear y ejecutar aplicaciones dentro de contenedores. Estos contenedores incluyen todo lo necesario para que la aplicación funcione correctamente.

## Relación entre GitHub y Docker
GitHub se utiliza para almacenar el código fuente y los archivos de configuración, como el Dockerfile. Docker utiliza estos archivos para construir imágenes y ejecutar contenedores de manera eficiente.
